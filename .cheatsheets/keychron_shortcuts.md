Keychron - Shortcuts
====================

```YAML
fn + 1/2/3: Switch bluetooth device (hold 4s to pair)
fn + tab  : Toggle the backlight on/off.
fn + a    : Next lighting effects.
fn + q    : Previous lighting effects.
fn + z    : Increase the backlight brightness.
fn + s    : Decrease the backlight brightness.
fn + t    : Increase the light effect speed.
fn + g    : Decrease the light effect speed.
```
